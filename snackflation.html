Expected site features include minigames, live or semi-live statistical predictions, simulations and emulations, neural networks that run in-browser, and more.

The new SSG is therefore necessary to support this. It will be able to generate new types of HTML5 and ECMAScript, permitting both static and dynamic widgets for various roles and capabilities. The new SSG will also support modern accessibility endeavors. It'll permit people using screenreaders and other automated or semiautomated interfaces to interact more easily with my web site. This will be accomplished by using HTML5 semantic components, as well as by using the ARIA accessibility metadata tags. It'll even be able to handle multilingual text. This will vastly increase the accessibility and usability of the site.

In summary, the web site is being remodeled to use a more modern CMS and SSG. The result of these new features will be to create a novel, more interesting experience for those who come to the site, while also increasing the amount of people who are capable of using this site.
