About the site

The site is generated using an SSG (static site generator). I write every page in a custom XML-based article format on my desktop computer. The SSG converts each page into HTML in a special folder. I can then simply use a git commit command to sync that folder to a GitHub repo. That GitHub repo then shows my page as a github.io site.

I'm not John Moneybags, so rather than paying money for a cool advanced SSG, I just wrote my own. It might not be as advanced as some SSGs, but it works just fine! The SSG's core is written in Python. It uses HTML templates to allow me to change the layout easily - that's how I am able to achieve homogeneity across the site. All CSS is located in one central CSS document. I don't use a preprocessor for my CSS yet, as it hasn't become necessary and I'm more proficient in raw CSS than in any preprocessors.
