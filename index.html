How to cheat at Wordle

Wordle was purportedly created by Josh Wardle as a fun word-guessing game. The rules are simple: You have six tries to guess a five-letter word. If a letter is guessed right, the letter gets colored green. If a letter is present in the word but it is not in that position, the letter gets colored yellow. Otherwise, the letter is colored gray. It's a conceptually simple game, but it has spawned a large amount of discussion over how it can be optimized.
