The new CMS will allow a greater variety of content to be hosted on this site. For instance, it'll allow for interactive demos. Expected site features include minigames, live or semi-live statistical predictions, simulations and emulations, neural networks that run in-browser, and more.

The new SSG is therefore necessary to support this. It will be able to generate new types of HTML5 and ECMAScript, permitting both static and dynamic widgets for various roles and capabilities. The new SSG will also support modern accessibility endeavors. It'll permit people using screenreaders and other automated or semiautomated interfaces to interact more easily with my web site. This will be accomplished by using HTML5 semantic components, as well as by using the ARIA accessibility metadata tags. It'll even be able to handle multilingual text. This will vastly increase the accessibility and usability of the site.

In summary, the web site is being remodeled to use a more modern CMS and SSG. The result of these new features will be to create a novel, more interesting experience for those who come to the site, while also increasing the amount of people who are capable of using this site.

About Snackflation

I built a fully automatic tool to predict market behavior and price indices of popular snack foods. It works by using a simple linear regression model to predict the future value of snacks such as potato chips. I regressed prices against the date and found that the date tends to have a statistically-significant impact upon price. This reflects well-known notions of inflation as predicted by many empirical models, which state that (during short runs) a market tends to be in an inflationary state rather than a deflationary state and all goods' prices will therefore rise over time, often at rates which are somewhat consistent.
